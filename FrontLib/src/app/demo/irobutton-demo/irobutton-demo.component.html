Button

It provides the same functionality as a native <input type="button"> enhanced with Material Design styling and animations.

Attributes Details :

  • title : String

    This is the component title shown after the component element. Also on click of this component you can toggle the component element.

  • color : String

    Set theme of component with color attribute

  • fill : Boolean

    Filled or outline button

  • isDisabled : Boolean

    to make button disabled

  • rounded : Boolean

    to make button rounded
